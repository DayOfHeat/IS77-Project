# Repository Structure

## Root

`README.md` - Final project report

`dependencies.txt` - List of python dependencies

`ProjectPlan.md` - Early plan for the project including research questions

`StatusReport.md` - Midpoint report including re-evaluation of project plan

`Snakefile` - Snakemake file that handles workflow automation

## /data

Contains raw and processed data used in the project. Contents are produced by running `Snakefile`.

### Raw Data

`fraud_raw.xlsx` - Excel sheet of fraud data fetched from [UK.gov](https://www.gov.uk/government/statistics/fraud-and-error-in-the-benefit-system-financial-year-2024-to-2025-estimates).

`labor_raw.xls` - Excel sheet of labor data fetched from [UK.gov](https://www.ons.gov.uk/employmentandlabourmarket/peopleinwork/employmentandemployeetypes/datasets/summaryoflabourmarketstatistics).

### Parsed Data

`fraud_dollars_parsed.csv` - Fraud data in dollars from `fraud_raw.xlsx` processed into a `.csv` file.

`fraud_percent_parsed.csv` - Fraud data as a percent of expenditures from `fraud_raw.xlsx` processed into a `.csv` file.

`fraud_percent_no_uc_parsed.csv` - Total benefits fraud data excluding Universal Credit as a percent of expenditures computed from `fraud_raw.xlsx`.

`unemployment_parsed.csv` - Unemployment data from `labor.xls` processed into a `.csv` file.

### Combined Data

`clean_fraud_dollars_unemployment.csv` - Data from `unemployment_parsed.csv` average into years instead of rolling 3-month periods and combined with `fraud_dollars_parsed.csv` with missing values imputed.

`clean_fraud_percent_unemployment.csv` - Data from `unemployment_parsed.csv` average into years instead of rolling 3-month periods and combined with `fraud_percent_parsed.csv` with missing values imputed.

`clean_fraud_percent_no_uc_unemployment.csv` - Data from `unemployment_parsed.csv` average into years instead of rolling 3-month periods and combined with `fraud_percent_no_uc_parsed.csv` with missing values imputed.

## /scripts

Contains python scripts used by `Snakemake`.

`fetch_data.py` - Fetches `fraud_raw.xlsx` and `labor_raw.xls` and places them in `/data`.

`parse_data_fraud.py` - Parses `fraud_raw.xlsx` into `fraud_dollars_parsed.csv`, `fraud_percent_parsed.csv`, and `fraud_percent_no_uc_parsed.csv`.

`parse_data_labor.py` - Parses `labor_raw.xls` into `unemployment_parsed.csv`.

`combine_and_clean_data.py` - Cleans and combines data from `parse_data_fraud.py` and `parse_data_labour.py`.

`analysis.py` - Produces statsmodel linear regression outputs in `/analysis` with combined data from `clean_and_combined_data.py`.

`make_graphs.py` - Produces graphs in `/graphs` with combined data from `clean_and_combined_data.py`.

## /analysis

Contains linear regression outputs from `statsmodel.api` generated by `Snakemake`.

`fraud_dollars_unemployment.txt` - Linear regression for total benefits fraud in dollars vs unemployment rate.  

`fraud_percent_unemployment.txt` - Linear regression for total benefits fraud as a percentage of expenditures vs unemployment rate.  

`fraud_percent_no_uc_unemployment.txt` - Linear regression for total benefits fraud excluding Universal Credit as a percentage of expenditures vs unemployment rate.  

## /graphs

Contains graphs generated by `Snakemake` using `matplotlib`.

`all_programs.png` - A graph of fraud in each benefit program as a percentage of expenditures over time.

`total_fraud.png` - A graph of total benefits fraud in dollars and UK unemployment rate over time and a graph of benefits fraud as a percentage of expenditures and UK unemployment rate over time.

`no_uc.png` - Total benefits fraud excluding Universal Credit as a percentage of expenditures and UK unemployment rate over time.
